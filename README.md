# 🌦️ 기상자료 보고서 생성기 Web App v3.0

기상청 **ASOS 일자료 CSV**를 업로드하면  
대화형 차트, 피벗테이블, Python 고품질 차트,  
Excel 및 PDF 보고서를 자동 생성하는 웹앱입니다.

Streamlit 기반으로 제작되었습니다.

---

# 🚀 주요 기능

## 📊 1. 대화형 차트 (Plotly)

- 관측소 선택 (다중 선택 가능)
- 기상요소 선택
- 집계단위 선택  
  - 일별  
  - 월별  
  - 연별
- 차트 유형  
  - 선 그래프  
  - 막대 그래프  
  - 복합 차트 (기온 + 강수량)

지원 요소:

- 평균기온
- 최고기온
- 최저기온
- 강수량
- 습도
- 풍속
- 일조시간
- 일사량
- 적설

---

## 📋 2. 동적 피벗 테이블

사용자가 직접 선택:

- 행(Row)
- 열(Column)
- 값(Value)
- 집계함수

지원 집계:

- 평균
- 합계
- 최대
- 최소
- 개수

결과:

- 화면 표시
- CSV 다운로드 가능

---

## 🐍 3. Python 고품질 차트

Matplotlib / Seaborn 기반

지원 차트:

1. 기온 시계열
2. 월별 강수량
3. 기온 Box Plot
4. 연×월 히트맵
5. 기온 vs 강수량 산점도

결과:

- PNG 다운로드 가능
- 보고서 활용 가능

---

## ⬇️ 4. 보고서 다운로드

자동 생성:

- Excel (.xlsx)
- PDF (.pdf)

포함 내용:

- 원본 데이터
- 피벗 테이블
- 요약 통계
